# React Navigation SPA Assignment

##  Project Overview
This project is a Single Page Application (SPA) built with React and Vite.  
It demonstrates navigation between multiple pages using React Router and includes both state management and API data fetching.

---

##  Features

###  Navigation (SPA)
- Built using React Router
- No page reloads between views
- Navigation bar links to different pages

###  Counter Page (State Management)
- Uses React `useState`
- Simple interactive counter
- Demonstrates state updates and UI re-rendering

###  Employees Page
- Displays a list of employees
- Uses reusable components (`EmployeeCard`)
- Data imported from a local file

###  Shopping Cart Page
- Add and remove items from cart
- Quantity updates dynamically
- Calculates total price

###  Weather Page (API Integration)
- Fetches real-time weather data from OpenWeather API
- Uses Axios for HTTP requests
- Displays weather information based on user input
